# 요구사항정의서 — AI 면접 코치 서비스

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | AI 면접 코치 |
| 목적 | 직군별 예상 질문 자동 생성 + 답변 피드백 제공 |
| 타깃 | 취준생, 이직 준비자 |
| 플랫폼 | 웹 앱 (MVP 기준) |

---

## 2. 기능 요구사항

### F1. 직군 선택 및 질문 생성
- 직군 입력 (개발/마케팅/기획/디자인 등)
- 경력 레벨 선택 (신입/3년/5년+)
- LLM 기반 예상 질문 10~20개 자동 생성

### F2. 답변 입력 및 피드백
- 텍스트 또는 음성 답변 입력
- STAR 구조 기반 피드백 자동 생성
- 항목별 점수 제공 (논리성/구체성/직무적합성)

### F3. 세션 관리
- 면접 히스토리 저장
- 약점 질문 재도전 기능

### F4. 리포트
- 세션 종료 후 종합 피드백 PDF 출력

---

## 3. 비기능 요구사항

| 항목 | 기준 |
|------|------|
| 응답속도 | 질문 생성 3초 이내 |
| 가용성 | 99.5% uptime |
| 보안 | 답변 데이터 암호화 저장 |
| 확장성 | 직군 카테고리 추가 용이한 구조 |

---

## 4. 제약사항

- MVP 개발기간: **6주 이내**
- LLM: OpenAI API 사용 (자체 모델 학습 없음)
- 음성 입력: MVP에서는 선택 기능 (필수 아님)
- 다국어: 한국어 단일 지원 (v1 기준)

---

## 5. 우선순위

| 순위 | 기능 | 근거 |
|------|------|------|
| P0 | 질문 생성 (F1) | 핵심 가치 |
| P0 | 텍스트 피드백 (F2) | 핵심 가치 |
| P1 | 세션 저장 (F3) | 재방문율 직결 |
| P2 | PDF 리포트 (F4) | 유료 전환 트리거 |
| P3 | 음성 입력 | 리소스 대비 효용 검토 필요 |

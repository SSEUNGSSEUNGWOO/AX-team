## MVP 범위

| 기능 | 담당 | 상태 |
|---|---|---|
| 직군 선택 → 질문 생성 | 수영 (AI+BE) | 설계 중 |
| 답변 텍스트 입력 | 주혁 (FE) | 대기 |
| 피드백 생성 (LLM) | 수영 | 설계 중 |
| 결과 저장/조회 | 수영 (BE) | 대기 |
| 기본 인증 | 수영 (BE) | 대기 |

> **MVP 제외 항목**: 음성, 영상, 비교 분석, 실시간 스트리밍

---

## AI 파이프라인 설계

```
[직군 입력]
     ↓
[질문 생성 프롬프트] → LLM → [질문 리스트]
     ↓
[사용자 답변 입력]
     ↓
[피드백 프롬프트] → LLM → [구조화 피드백]
     ↓
[DB 저장] → [결과 반환]
```

> 프롬프트는 유진 크리틱 통과 전 운영 반영 금지.

---

## 자동화 항목

| 항목 | 도구 | 시점 |
|---|---|---|
| 프롬프트 버전 관리 | Git + YAML | MVP부터 |
| API 테스트 자동화 | pytest | M2 완료 후 |
| 프롬프트 품질 평가 | LLM-as-Judge | M3 QA |
| 배포 파이프라인 | GitHub Actions | M3 |
| 에러 알림 | Sentry | 배포 전 |

# 쿠팡 가격 모니터링 툴 — 요구사항정의서

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Coupang Price Watcher |
| 목적 | 쿠팡 상품 URL·목표가 등록 후 주기적 가격 감지 및 콘솔 알림 자동화 |
| 대상 사용자 | 내부 운영팀 (단일 사용자 CLI 환경) |
| 개발 언어 | Python 3.10+ |
| 배포 환경 | 로컬 CLI / 추후 서버 스케줄링 가능 구조 |

## 2. 기능 요구사항

### 2-1. 상품 관리 (CRUD)

| ID | 기능 | 설명 | 우선순위 |
|----|------|------|----------|
| F-01 | 상품 등록 | URL + 목표가 + 별칭 입력 → DB 저장 | **P0** |
| F-02 | 상품 목록 조회 | 등록된 전체 상품·목표가·현재가 출력 | **P0** |
| F-03 | 상품 수정 | 목표가·별칭 변경 | **P1** |
| F-04 | 상품 삭제 | URL 기준 단건 삭제 | **P1** |

```
# 등록 예시 CLI
python watcher.py add --url "https://www.coupang.com/vp/products/..." --target 29000 --alias "무선마우스"
python watcher.py list
python watcher.py update --id 3 --target 25000
python watcher.py delete --id 3
```

### 2-4. 데이터 저장 (DB 스키마)

**SQLite 단일 파일 (`watcher.db`)**

```sql
-- 상품 테이블
CREATE TABLE products (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    url         TEXT    NOT NULL UNIQUE,
    alias       TEXT,
    target_price INTEGER NOT NULL,
    is_active   INTEGER DEFAULT 1,          -- 0: 비활성
    alerted_at  TEXT,                        -- 마지막 알림 시각 (ISO8601)
    created_at  TEXT    DEFAULT (datetime('now','localtime')),
    updated_at  TEXT    DEFAULT (datetime('now','localtime'))
);

-- 가격 이력 테이블
CREATE TABLE price_history (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    product_id  INTEGER NOT NULL REFERENCES products(id),
    price       INTEGER NOT NULL,
    fetched_at  TEXT    DEFAULT (datetime('now','localtime')),
    status      TEXT    DEFAULT 'ok'         -- 'ok' | 'parse_error' | 'network_error'
);
```
